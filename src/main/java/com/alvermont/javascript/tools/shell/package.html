Contains a slightly modified version of the Mozilla Rhino Javascript Shell. The modifications are mostly to make it a better citizen as part of a Swing application. This includes changes to when the thread is started and removal of the calls to System.exit(). The code to change the UI look and feel has also been removed. It is assumed that the main application will handle this.
